# Интеграция CRM с сервисом MadeTask

Введение: данный документ описывает требования интеграции CRM и MadeTask для автоматизации управления задачами и проведения выплат исполнителям  

Цели:  
- учесть всех исполнителей (и внутренних, и внешних) в одной системе
- автоматизировать выплаты дизайнерам при выполнении задач
- снизить нагрузку на бухгалтерию (избежать ручных переводов)
-  
Задачи:   
-	реализовать интеграцию CRM с MadeTask
-	автоматизировать процесс проведения выплат исполнителям
-	обеспечить синхронизацию данных между системами
-	автоматизировать процесс создания и управления задачами

1. Анализ текущих процессов
   
Текущий процесс:  
- менеджер ставит задачу дизайнерам через мессенджеры;
- чтобы проверить выполненную задачу, менеджеру необходимо найти её в чат;
- оплата производится вручную. Бухгалтерия переводит деньги через СБП, сверяя реквизиты в Excel.
  
Проблемы и причины  
- задачи теряются, потому что нет единого трекера и все задачи передаются через мессенджеры;
- много времени уходит на перевод денег, потому что бухгалтер вручную переводит деньги через СПБ, сверяя реквизиты в Excel;
- риск, что платеж может уйти не тому получателю, за счет ручного ввода данных;
- нет прозрачности в процессах, так как для проверки исполнения платежа необходимо ждать отчет из банка.
